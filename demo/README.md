# JWT Authentication using Spring Boot

## Features
- Login with username/password
- JWT token generation
- Protected routes
- Token validation

## API Endpoints
- POST /api/auth/login
- GET /api/auth/protected

## How to Run
1. mvnw clean install
2. mvnw spring-boot:run

## Postman Steps
1. Login → get token
2. Use token → access protected route
